Minimal base

At minimum, I rely only on these software that runs on all major platforms 1 :

  • A web browser with adblock (eg: Brave, Safari)
  • Nix (reproducible development environment)
  • Syncthing (file sync)
  • Pass with GPG (encrypted secrets & passwords)
  • VSCode

Everything else is optional. This allows me to use any OS as I see fit, as well move between them as the situation requires, without developing dependency on one. Hence, ‚Äúminimal base‚Äù of software use/dependency.

Inspiration

Inspiration for this stance comes from a combination of Intention trumps convenience and the suckless philosophy.
